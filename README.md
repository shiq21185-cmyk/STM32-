# 智能语音药盒系统

基于STM32F103+FreeRTOS的多功能智能药盒，集成多传感器感知、离线语音交互、多模态提醒、数据上传等功能，为用户提供智能化服药管理解决方案。

## 📦 核心功能

- **智能服药提醒**：支持多组闹钟设置，到点触发语音+灯光+舵机开盖多模态提醒
- **取药动作检测**：重量传感器+光敏传感器双重校验，准确识别取药行为，避免漏服误报
- **空盒报警机制**：自动检测药盒剩余药量，余量不足时主动提醒用户补药
- **全离线语音控制**：无需手机APP，纯离线语音即可完成闹钟配置、功能设置等全部操作
- **蓝牙数据上传**：支持服药数据、设备状态上传到手机端，便于健康管理
- **低功耗运行**：传感器周期性唤醒，非关键任务低优先级调度，延长设备续航

## ✨ 创新设计特点

### 1. 融合感知与高可靠容错技术
采用重量传感器+光敏传感器多模态融合算法，双重校验取药动作，大幅降低误报概率；搭配任务看门狗监控、MQTT断线自动重连、蓝牙超时回退等多重容错机制，全链路保障系统运行可靠性。

### 2. 全离线语音交互与无阻塞调度技术
支持完整的离线语音设置流程，无需依赖手机APP即可独立完成闹钟、服药提醒等全部功能配置；基于FreeRTOS抢占式多任务调度机制，实现任务的实时中断与恢复，语音交互全程无阻塞，操作响应流畅无卡顿。

### 3. 分级精准提醒与低功耗优化技术
采用分级提醒策略，闹钟提醒与空盒报警使用差异化语音ID与LED闪烁频率，用户可快速区分告警类型；搭载传感器周期性唤醒机制，非关键任务配置低优先级调度，在保证功能完整性的前提下大幅降低整机功耗。

## 📖 使用说明

### 开机
供电开机，OLED显示欢迎界面，系统自动完成初始化

### 设置闹钟
- **语音方式**：说出"设置闹钟"，根据语音提示依次设置时、分即可
- **按键方式**：短按功能键进入设置模式，上下键调整时间

### 服药提醒
闹钟到达设定时间后，语音播报提醒，LED闪烁，舵机自动打开药盒盖

### 取药确认
取出药物后，系统自动检测重量变化+光敏状态，确认取药完成后关闭提醒

### 空盒提醒
当药盒内药物余量不足时，系统自动播报"请补充药物"提示

## 🔌 引脚连接

所有模块必须与 STM32 共地。GPIO 电平为 3.3 V；若外设 TX 输出为 5 V，需先进行电平转换。

| 功能 | STM32 引脚 | 接线说明 |
| --- | --- | --- |
| HX711 时钟 / 数据 | PB14 / PB15 | 分别连接 `SCK` / `DOUT` |
| DHT11 数据 | PA5 | 数据线接 PA5，并按模块要求配置上拉 |
| 光敏模块 | PB4 | 模块数字输出接 PB4 |
| 指示灯 | PA0 | 低电平点亮；普通 LED 按“3.3V → 限流电阻 → LED 正极 → PA0”连接 |
| 舵机 PWM | PB3（TIM2_CH2） | 舵机信号线接 PB3；舵机电源建议独立 5V，并与主控共地 |
| 按键 | PB8 / PB9 / PB6 / PB7 | 对应页面、确认、减小、增大；程序使用下拉输入，高电平为按下 |
| OLED 软件 I²C | PA12 / PA11 | 分别连接 SCL / SDA |
| HC-06 | PA9 / PA10（USART1） | STM32 TX 接模块 RX，STM32 RX 接模块 TX |
| ESP8266 | PA2 / PA3（USART2） | STM32 TX 接模块 RX，STM32 RX 接模块 TX；复位控制使用 PA1 |
| 小 R 语音模块 | PB10 / PB11（USART3） | 分别连接模块 RX / TX |

## 💾 闹钟设置断电保存

三组闹钟的小时、分钟和启用状态会保存到 STM32 内部 Flash 的专用页。语音或蓝牙修改闹钟后会保存；按键编辑时仅在退出编辑或切换页面后保存一次，避免连续调节时频繁擦写 Flash。

该功能只保存闹钟配置，不保持当前系统时间。重新上电后仍应通过蓝牙或其他方式校准时间。

## 💻 开发环境

- **集成开发环境**：Keil MDK5
- **编译工具**：ARM Compiler v5
- **调试工具**：J-Link/ST-Link
- **烧录方式**：SWD接口烧录

### 编译与烧录步骤

1. 使用 Keil MDK5 打开 `Poject.uvprojx`。
2. 选择 `Target 1`，确认芯片型号为 STM32F103C8 系列。
3. 编译工程；首次运行请确认各模块接线和供电正确。
4. 通过 ST-Link/J-Link 的 SWD 接口下载程序。
5. 上电后先完成 HX711 去皮和时间校准，再设置闹钟进行测试。
